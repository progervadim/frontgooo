Пример:

Для того, чтобы внедрить данный фон себе на страницу, нам потребуется базовые знания HTML, CSS, а так же JS. Давайте рассмотрим по подробнее.
Первым делом я оставлю весь проект сдесь:

See the Pen QWjqdoq by progervadim (@progervadim) on CodePen.

Теперь разберем код подробнее:

body {
                    margin: 0;
                    background-color: #17182f;
                }
                    canvas {
                        display: block;
                        vertical-align: bottom;
                        }
                        #particles-js {
                        position: absolute;
                        width: 100%;
                        height: 100%;
                        }

margin: 0; - Уберает отступы
background-color: #17182f; - Задает цвет фона страницы, в нашем случае это темно-синий цвет
display: block; - Задаем значение подобно блоку
vertical-align: bottom; - Выравнивает основание текущего элемента по нижней части элемента строки
Далее прописывает значения нашему основному классу:
position: absolute; -Абсолютное позиционирование
Так же задаем ширину и высоту на 100%
